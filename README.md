Frontend Components Overview

This project includes several React components under src/components, each responsible for a specific visualization or interaction feature.

AuthorNetwork.tsx

Renders the author collaboration network using D3, visualizing co-authorship relationships as an interactive graph.

ChatWithLLM.tsx

Implements a chat interface that allows users to interact with the LLM, send natural-language queries, and receive structured analytical results or visualizations.

CitationGraph.tsx

Displays a force-directed citation network where papers are represented as nodes and citations as links, enabling interactive exploration through D3.js.

CitationGraphRadialBundled.tsx

Provides a radial edge-bundling version of the citation network, offering a circular layout that highlights citation clusters and structural patterns.

T2Dashboard.tsx

Acts as the main visualization dashboard for Project 2, integrating Vega-Lite charts and LLM-generated insights into a unified interface.
